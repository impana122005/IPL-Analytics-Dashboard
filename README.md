# IPL Analytics Dashboard 📊

This project is built using Power BI to analyze IPL data (2008–2024).

## Features:
- Team performance analysis
- Toss impact analysis
- Top batsmen & bowlers
- Season-wise winners
- Venue analysis

## Tools Used:
- Power BI
- DAX
- Excel Dataset
